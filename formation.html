BTS SIO

Actuellement en BTS SIO (Systèmes informatiques aux organisations), au lycée Gaston Berger à Lille, j'ai eu l'occasion de découvrir de nombreuses technologies informatiques dans l'objectif de me spécialiser dans le domaine.
J'y ai pris l'option SLAM (Solutions Logicielles et Applications Métier)

Ce diplome m'a permis d'apprendre de nombreuses choses, que ce soit en développement, en réseau ou bien en cybersécurité.


Ce que j'ai appris en développement :

  • Javascript côté navigateur
  • PHP avec intégration d'une base de donnée MySQL
  • Les bases du Java
  • Gestion de base de données
  • Structure d'un site web avec HTML et design grâce au CSS
  • La mise en place d'un client lourd grâce au C# et à Java
  • Mettre en place une application mobile grâce à Flutter et Dart


    • Ce que j'ai appris en réseau :

  • Créer des machines virtuels
  • Mettre en place un Active Directory sur Windows
  • Installer des RAIDS
  • Mise en place d'un serveur web avec continuité de service


    • Ce que j'ai appris en cybersécurité :

  • Créer une machine Kali-Linux pour déchiffrer des mots de passe
  • Le chiffrement des mots de passe (de manière symétrique ou asymétrique)
  • Le hashage d'une phrase / d'un mot de passe
  • La sécurisation d'un site web, par le code ou bien par la protection des serveurs
